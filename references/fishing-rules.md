# 钓鱼分析规则库

本文档包含钓鱼决策智能体的核心分析规则，各模块调用时需严格遵循。

## 一、溶解氧分析规则

### 1.1 基础溶氧计算（季节水温）

| 季节 | 基础溶氧范围（mg/L） | 说明 |
|------|---------------------|------|
| 冬季 | 10-13 | 低温高溶氧 |
| 春秋 | 8-10 | 适宜温度 |
| 夏季 | 6-8 | 高温低溶氧 |
| 盛夏（>30℃） | 4-6 | 极高温极低溶氧 |

### 1.2 气压修正系数

| 气压范围（hPa） | 修正系数 | 说明 |
|----------------|---------|------|
| >1020 | +15% | 高压稳定 |
| 1010-1020 | 0% | 正常 |
| 990-1010 | -15% | 低压 |
| <990 | -30% | 特低压 |

**气压趋势修正**：
- 快速上升：+10%
- 快速下降：-20%
- 稳定：0%

### 1.3 天气修正系数

| 天气类型 | 风力条件 | 修正系数 |
|---------|---------|---------|
| 晴天 | ≥2级 | +10% |
| 晴天 | 无风 | -5% |
| 阴天 | 任意 | -5% |
| 中到大雨 | 任意 | +15% |
| 雷阵雨前闷热 | 任意 | -25% |

### 1.4 河道类型修正系数

| 河道类型 | 修正系数 | 说明 |
|---------|---------|------|
| S型（山区溪流） | ×1.2 | 流动性好 |
| R型（平原曲流） | ×1.0 | 标准 |
| L型（大型干流） | ×0.9 | 流速慢 |
| U型（城市河段） | ×0.6 | 污染影响 |
| W型（草型河道） | 白天×1.1 / 夜间×0.7 | 光合作用 |

### 1.5 海拔修正

| 海拔高度 | 修正系数 |
|---------|---------|
| >1500m | ×0.7~0.9 |
| ≤1500m | ×1.0 |

### 1.6 溶氧评级标准

| 溶氧量（mg/L） | 评级 | 说明 |
|---------------|------|------|
| >7 | 高 | 鱼类活跃 |
| 5-7 | 中 | 正常作钓 |
| 3-5 | 低 | 需选择富氧区 |
| <3 | 极低 | 不建议作钓 |

## 四、河道类型修正规则

### 4.1 河道类型识别

**判断依据**：
1. 地理位置特征（城市中心/郊区/野外）
2. 水文特征（流速、水质、水草）
3. 人造设施（桥梁、码头、排污口）

**API辅助**：
- 通过逆地理编码判断区域类型
- 结合卫星地图判断河道特征

### 4.2 修正参数对照表

| 参数 | 野外河道（wild） | 城市河道（urban） |

| 鱼类活跃度修正 | ×1.0 | ×0.9（长期亚健康） |
| 有效地形类型 | 全9类 | 仅P1(慎选)、P2、P5、P9、P4 |

## 五、鱼种应激规则

### 5.1 应激源识别

| 应激源 | 触发条件 | 影响 |
|--------|---------|------|
| 温差突变 | >5℃ | 停口、窗口期缩短 |
| 气压骤降 | 下降>5hPa/3h | 鱼上浮、停口 |
| 溶氧极低 | <3mg/L | 暗浮、无法存活 |
| 水质浑浊 | 能见度<30cm | 鳃受损、停口 |

### 5.2 应激行为速查表

详见 `data/fish-preferences.json` 中的应激行为字段

### 5.3 应激应对策略

| 应激类型 | 应对策略 |
|---------|---------|
| 温差应激 | 逗钓、活饵、缩小饵料 |
| 气压应激 | 选择富氧区、钓浅、飞铅 |
| 溶氧应激 | 寻找入水口、出水口、增氧机旁 |
| 浑浊应激 | 浓腥饵、亮片、噪音饵 |

## 六、综合评分规则

### 6.1 分项系数计算

#### 温度系数
```
温度系数 = 高斯函数(实际水温, 鱼种最适水温范围)
- 在适温区间内：100分
- 每偏离1℃：-5分
- 超出摄食范围：0分
```

#### 溶氧系数
```
溶氧系数 = min(实际溶氧 / 鱼种最适溶氧 × 100, 100)
- 实际溶氧 < 1mg/L：0分
- 封顶：100分
```

#### 地形系数
```
地形系数 = 
- 该地形在鱼种偏好列表中：90-100分
- 该地形不在偏好列表中：30-50分
```

#### 食物系数
```
食物系数 = 食物潜力值 × 食物类型匹配度
- 食物潜力值：极高100 / 高75 / 中50 / 低25
- 食物类型匹配度：0.5-1.0
```

### 6.2 加权综合得分

**标准权重**：
```
综合得分 = 温度系数×0.35 + 溶氧系数×0.35 + 地形系数×0.2 + 食物系数×0.1
```

**特殊鱼种权重调整**：
- 草鱼：温度0.3、溶氧0.3、地形0.2、食物0.2
- 鳜鱼：温度0.35、溶氧0.35、地形0.25、食物0.05
- 翘嘴：温度0.35、溶氧0.35、地形0.2、食物0.1

### 6.3 应激修正系数

| 应激等级 | 修正系数 | 说明 |
|---------|---------|------|
| 无应激 | 1.0 | 正常 |
| 轻度应激 | 0.8 | 温差3-5℃ |
| 中度应激 | 0.6 | 温差5-8℃或气压下降3-5hPa |
| 重度应激 | 0.5 | 温差>8℃或气压下降>5hPa |

### 6.4 出钓建议等级

| 综合得分 | 等级 | 建议 |
|---------|------|------|
| >75 | 强烈建议 | 黄金窗口 |
| 60-75 | 可以出钓 | 注意选择最佳钓点 |
| 40-60 | 勉强可钓 | 需要特殊技巧 |
| <40 | 不建议 | 改日或夜钓 |

## 八、容错与默认值

### 8.1 字段缺失处理

| 缺失字段 | 默认值 | 处理方式 |
|---------|--------|---------|
| 气压 | 1013hPa | 提示用户"气压未知，使用标准大气压" |
| 天气 | 晴 | 提示用户"天气未知，按晴天计算" |
| 河道类型 | 野外 | 提示用户"河道类型未知，按野外计算" |
| 目标鱼种 | 鲫鱼 | 提示用户"目标鱼种未指定，默认鲫鱼" |

### 8.2 异常值处理

| 异常情况 | 处理方式 |
|---------|---------|
| 气压<900hPa 或 >1050hPa | 提示数据异常，要求用户确认 |
| 温度<-20℃ 或 >45℃ | 提示数据异常，要求用户确认 |
| 经纬度格式错误 | 提示格式错误，要求重新输入 |
