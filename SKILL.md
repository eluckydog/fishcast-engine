# 🎣 钓鱼佬永不空军 — FishCast Engine

> "今天不空军！" — WorkBuddy/QClaw 适配版

## 能力

本 Agent 可接收自然语言查询，自动完成以下流程：

```
用户输入 → 解析位置/时间 → 查天气 → 查水体 → 评分 → 输出策略
```

## 触发词

- "钓鱼""钓点""出钓""空军""鱼情"
- "推荐钓点""附近能钓鱼吗""今天适合钓鱼吗"

## 使用方式（三选一）

### 方式一：自然语言查询（推荐）

用户说一句话，Agent 自动解析：

```
"上海市崇明区 2026年6月22日下午3点 推荐方圆5公里内的最佳钓点"
"北京密云水库今天能钓鱼吗"
"广州珠江边晚上8点钓什么鱼好"
```

### 方式二：指定参数

```
fishcast 鲤鱼 --temp 26 --pressure 1015 --weather 晴
```

### 方式三：Web 交互（打开 Pages）

```
打开 https://eluckydog.github.io/fishcast-engine/
```

## 执行流程

### Step 1 — 解析用户输入

从用户输入中提取：地点（城市/区/地标）、时间（日期+时段）、目标鱼种（可选）、搜索半径（默认3km）

### Step 2 — 自动获取数据

| 数据 | 获取方式 | 备注 |
|:-----|:---------|:-----|
| 经纬度 | 调高德/腾讯/百度逆地理编码（优先），或内置坐标库 | 免费API额度足够 |
| 天气 | Open-Meteo API（免费无Key）或和风天气API（推荐中国区） | Open-Meteo无需Key |
| 水体 | Overpass API (OpenStreetMap) | 免费无限制 |
| 鱼种 | 内置 69 种鱼数据 | 本地JSON |

### Step 3 — 六部评分

对每个候选水体计算综合评分，输出 TOP 3 推荐。

### Step 4 — 输出格式

按以下结构返回给用户：

```
📍 位置：XX（经纬度）
📅 时间：XXXX年XX月XX日 XX:XX

🌤️ 天气：
  气温: XX°C | 气压: XXX hPa | 湿度: XX%
  天气: XX | 风速: XXkm/h XX风

💧 附近水体（半径Xkm内）:
  - XXX (类型) — 距离 X.Xkm

🎯 推荐钓点 TOP X:
  🥇 XXX
    目标鱼种：XX
    评分：XX/100 — 「黄金出钓窗口」
    🪝 钩型+线组推荐
    🥫 打窝策略
    📖 钓法技巧

💡 综合建议：
  - 最佳窗口
  - 注意事项
```

## 配置（可选）

如需使用中国区更精准的 API，配置环境变量：

```bash
# 和风天气（推荐）
export QWEATHER_KEY="your_key"

# 高德地图
export AMAP_KEY="your_key"
```

未配置时自动降级为 Open-Meteo + OSM（免费无Key）。

## 示例

**用户**："上海市崇明区明天早上钓鱼推荐"

**Agent 执行**：
1. 解析 → 地点:崇明区(31.62,121.40) | 时间:次日6:00 | 半径:3km
2. 天气 → Open-Meteo: 26°C, 1008hPa, 多云
3. 水体 → Overpass: 东平河/崇明环岛运河/北横引河
4. 评分 → 每条水体对常见鱼种评分
5. 输出 → TOP 3 钓点 + 钓具 + 打窝 + 技巧

**输出**：

```
📍 上海市崇明区 (31.62°N, 121.40°E) | 🌅 清晨窗口

🌤️ 26°C · 1008hPa · 多云 · 东南风12km/h

🎯 推荐钓点：
  1️⃣ 东平河南段 (1.2km) — 82分
     🐟 鲫鱼·鲤鱼·鳊鱼·翘嘴
     🪝 伊势尼5-7# + 综合竿4.5m
     🥫 玉米+颗粒打窝
  2️⃣ 北横引河中段 (3.5km) — 74分
     🐟 草鱼·鲢鳙·鳊鱼
     🪝 伊势尼6-8# + 综合竿5.4m
  3️⃣ 崇明环岛运河 (2.8km) — 65分
     🐟 鲫鱼·白条·黄辣丁
     🪝 袖钩3-5# + 鲫鱼竿3.6m

💡 建议带3.6m和5.4m两根竿，兼钓近远
```
