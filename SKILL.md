### Step 3 — 综合评分

对每个候选钓点计算：

```
温度系数 = 100 - max(0, |实际水温-最适水温均值| × 5)  [封顶100，超出摄食范围=0]
溶氧系数 = min(实际溶氧 / 鱼种最适溶氧 × 100, 100)    [实际溶氧<1mg/L=0]
地形系数 = 鱼种偏好地形? 90-100 : 30-50
食物系数 = 食物潜力分 × 食物类型匹配度
综合得分 = 温度系数×0.35 + 溶氧系数×0.35 + 地形系数×0.2 + 食物系数×0.1
如触发应激 → 综合得分 × 0.5~0.8
```

## 数据库说明

### 鱼种偏好库（`data/fish-preferences.json`）
- **覆盖鱼种**：鲫鱼、鲤鱼、草鱼、鳜鱼、翘嘴、鲶鱼、罗非鱼、青鱼、鳊鱼、黄颡鱼、马口、白条、鲢鱼、鳙鱼、鲮鱼、乌鳢、鳡鱼、大口黑鲈、虹鳟、蓝鳃太阳鱼、巴西鲷、梭鲈、欧洲鳊、丁鱥、革胡子鲶、斑点叉尾鮰、淡水白鲳、鳗鲡（共28种）
- **字段**：最适水温、摄食水温范围、最适溶氧、耐低氧能力、偏好地形、食物类型、钓法、饵料、应激触发条件

### 地形类型库（`data/terrain-types.json`）
- P1 入水口 / P2 出水口 / P3 回水湾 / P4 深浅交界 / P5 倒树沉船石堆
- P6 水草区 / P7 沙石浅滩 / P8 泥底腐殖质 / P9 桥墩码头
- 每种地形含：食物潜力、局部溶氧修正、局部水温修正、适用鱼种

### 河道修正参数（`data/river-correction.json`）
- **城市河道**：溶氧系数×0.6~0.8，夏季水温+2~5℃，有效地形仅P1(慎)/P2/P4/P5/P9
- **野外河道**：溶氧系数×1.0，全9类地形均有效

## 评分标准

| 综合得分 | 等级 |
|---------|------|
| >75 | 强烈建议（黄金窗口）|
| 60-75 | 可以出钓 |
| 40-60 | 勉强可钓（需特殊技巧）|
| <40 | 不建议（改日或夜钓）|

## 示例对话

**用户**："杭州千岛湖，明天钓鲤鱼，气温26度，气压1015hPa，晴天"

**执行**：
1. 提取参数（野外河道，夏天，26℃，1015hPa，鲤鱼）
2. 户部：溶氧=7×1.0×1.0×1.0=7mg/L（高）；水温=26℃，活跃度75%
3. 工部：野外河道全地形有效，P1入水口食物潜力100溶氧+40%
4. 吏部：鲤鱼最适水温15-28℃，溶氧4.5mg/L，偏好P3/P4/P8/P9
5. 刑部：无显著应激风险
6. 礼部：综合评分→输出策略
